# Name the CLI agentic-afk

The command-line entrypoint will be named `agentic-afk`. This is more specific than `afk`, which is too generic and collision-prone, while avoiding `dioxus-agentic-afk` because Dioxus is an implementation detail of the dashboard rather than the product boundary.

**Considered Options**

- `afk`
- `dioxus-agentic-afk`
- `agentic-afk`
