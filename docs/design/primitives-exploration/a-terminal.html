Terminal Brutalist

Tone. tmux / k9s / lazygit. The control plane is a power-user instrument and looks like one. Monospace everywhere. Square corners. One sodium-amber accent on near-black. No gradients, no rounding, no easing — only blink, step, and cut.

Type. IBM Plex Mono 300/400/500. Display tracked +0.06em, eyebrow tracked +0.22em uppercase, body untracked.
Density. 4px grid, 1px hairline rules, dashed rules for soft seams, ASCII corner glyphs.
Motion. 0.6–1.4s blink/step cycles. No bezier curves. Skeletons type themselves.
